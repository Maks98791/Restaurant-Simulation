# Restaurant-Simulation
Simulation of restaurant working with threads, mutexes and shared resources with ncurses visualization.

## Technologies
Project is created with:
* C++  version: 17

## Features
Project contains:
* Ncurses visualisation
* Resolved deadlock
* Safety critical sections
* Lack of data race
* Atomic types
